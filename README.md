# Guidelines for contributors

```
git switch your-branch-name
Do some changes
git add --a
git commit -m "your-message"
git switch main
git pull
git switch your-branch-name
git fetch
git merge main
git push
```

Head towards your git repository and create a pull request. Make sure that you are in the right branch.
